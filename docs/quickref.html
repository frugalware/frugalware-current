= Quick reference == Informations * Package management: pacman-g2 (command line) * Linux kernel 2.6 (no 2.4 support) * The latest documentation is http://frugalware.org/docs[here]. * Hardware requirements and list of supported architectures are in the Installation section of the documentation. == Features * Stable releases every 6 months * Security support for stable releases * Text mode installation * Optional graphical installation * Offline installation, netboot install supported * Prebuilt CD/DVD/USB hybrid, TFTP images are available * Localization supported whereever it's possible * About 5000 source packages and (as of March 2011) 6000 binary packages supported.
